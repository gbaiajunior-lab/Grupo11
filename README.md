# Projeto Integrador: Sistema de Gestão

Este repositório reúne os artefatos do projeto integrador do grupo, contendo documentos, códigos-fonte e protótipos desenvolvidos de forma colaborativa pelos integrantes.

## Estrutura do Repositório

```
projeto-integrador-sistema-gestao/
 ┣ docs/
 ┃ ┣ diagrama_caso_uso.png
 ┃ ┣ diagrama_classes.png
 ┃ ┗ prototipos/
 ┣ src/
 ┣ README.md
 ┗ LICENSE
```

- **docs**: Diagramas e protótipos visuais
  - `diagrama_caso_uso.png`: Diagrama de caso de uso
  - `diagrama_classes.png`: Diagrama de classes
  - `prototipos/`: Protótipos desenvolvidos para o sistema

- **src**: Código-fonte Java (se aplicável)

- **README.md**: Documentação do projeto

- **LICENSE**: Licença do projeto
